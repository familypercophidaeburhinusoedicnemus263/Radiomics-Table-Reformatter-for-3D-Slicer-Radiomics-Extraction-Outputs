## 🔍 What This Software Does

This tool helps you take output files from 3D Slicer radiomics feature extraction and change their format. Normally, the radiomics outputs come as CSV files that are hard to use directly with machine learning tools. This software rearranges those CSV files into sample-by-feature tables. These tables make it easier to analyze your data with machine learning or other statistical software.

You don’t need programming knowledge to use the software. It works in batch mode, so you can reformat many files at once. The goal is to save time and reduce errors you might make when handling these files manually.

## 🧾 About the Data Format

The tool works with CSV files that contain radiomics features on one axis and cases or image samples on the other. The original 3D Slicer output files usually list features row-wise or column-wise in a way that can be difficult to analyze. The software rearranges rows and columns, so each row represents one sample, and each column shows a feature.

This format is a common starting point when doing machine learning, data visualization, or statistical testing.
